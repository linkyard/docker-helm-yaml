# docker-helm-yaml

Docker image the contains kubernetes helm, git and utils for YAML and JSON processing.

## Contains

- helm (version 2.17.0)
- git
- jq
- spruce
- curl
- wget
- bash


## Usage

`docker run ghcr.io/linkyard/helm-yaml`
